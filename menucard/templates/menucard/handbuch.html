Einloggen

Der Benutzer kann sich über eine Eingabemaske in seinem Account einloggen. Hierfür muss der Benutzer seinen Benutzernamen und das Passwort eingeben. Durch Klicken auf Login wird der Benutzer in seinem Account eingeloggt und wird auf sein persönliches Dashboard weitergeleitet.

Produkt anzeigen

Auf dem Menükarten-Dashboard hat der Benutzer Zugriff auf sechs Unterschiedliche Bearbeitungsmöglichkeiten der Menükarte. Um ein Produkt anzulegen, muss der Benutzer auf Anzeigen der jeweiligen Sparte klicken. Durch den Klick auf Anzeigen wird der Benutzer auf die jeweilige Sparte weitergeleitet. Der Benutzer hat die Option

  • ein neues Produkt anzulegen,
  • ein existierendes Produkt zu bearbeiten,
  • ein existierendes Produkt aus der Menükarte zu löschen und
  • nach einem speziellen Produkt zu suchen, indem die Filter-Funktion benutzt wird.

Die Daten der bereits angelegten Produkte werden in einer Tabelle dargestellt. Der Benutzer erhält so eine Übersicht über die angelegten Produkte.

Produkt anlegen

Um ein Produkt anzulegen muss der Benutzer Vorspeise hinzufügen klicken. Daraufhin wird der Benutzer auf eine separate Seite weitergeleitet. Zu sehen ist eine Eingabemaske und ein Informationsfeld, dass sämtliche Zusatzstoffe und Allergene mit den jeweiligen Abkürzungen aufzeigt.
Damit der Benutzer ein Produkt anlegen kann, muss er die Pflichtfelder, die mit einem Sternchen (*) markiert sind ausfüllen. Um das Produkt anzulegen muss der Benutzer auf Speichern klicken.Durch einen Klick auf Abbrechen wird der Vorgang abbgebrochen und der Benutzer wird zurück auf die Übersicht der jeweilgen Sparte weitergeleitet.

Produkt löschen

Möchte der Benutzer ein Produkt aus der Menükarte entfernen bzw. löschen, muss er zunächst auf der Übersichtsseite der Sparte auf das (Mülltonnen-Symbol) klicken. Daraufhin wird der Benutzer auf die Löschen Seite weitergeleitet, wo der Benutzer die Daten des Produkts nochmal vor dem löschen überprüfen kann. Durch eine Klick auf Löschen wird das Produkt aus der Menükarte entfernt. Ein Klick auf Abbrechen beendet den Vorgang und der Benutzer wird zurück auf die Übersicht der jeweilgen Sparte weitergeleitet.

Produkt bearbeiten

Möchte der Benutzer ein Produkt aus der Menükarte bearbeiten, muss er zunächst auf der Übersichtsseite der Sparte auf das (Stift-Symbol) klicken. Daraufhin wird der Benutzer auf die Bearbeiten Seite weitergeleitet, wo der Benutzer die Daten des Produkts bearbeiten kann. Durch eine Klick auf Speichern werden die modifizierten Veränderungen übernommen und in der Menükarte aktualisiert. Ein Klick auf Abbrechen beendet den Vorgang und der Benutzer wird zurück auf die Übersicht der jeweilgen Sparte weitergeleitet.

Profil bearbeiten und Passwort ändern

Möchte der Benutzer seine persönlichen Daten ändern, muss er zunächst in der Navigationsleiste auf seinen Benutzernamen klicken. Daraufhin erscheint eine Dropdown-Liste mit zwei Optionen – Profil und Logout. Hier klickt der Benutzer Profil und es erscheint eine neue Seite mit den Daten des Benutzers. In der Eingabemaske kann der Benutzer nun seine Daten verändern und durch einen Klick auf Speichern die Daten erfassen oder durch einen Klick auf Abbrechen den Vorgang beenden.

Um das Passwort zu ändern muss der Benutzer auf Passwort ändern klicken. Anschließend öffnet sich ein Modal-Fenster, wo der Benutzer in einer Eingabemaske zunächst sein aktuelles Passwort eingeben muss. Anschließend gibt der Benutzer sein neues Passwort an und bestätigt in letzten Eingabefeld das neue Passwort und kann die Änderung durch das Klicken auf Speichern übernehmen. Durch einen Klick auf Abbrechen wird der Vorgang beendet.

Erfasste Corona-Anmeldung anzeigen

Damit der Benutzer die erfassten Daten seiner Kunden anzeigen lassen kann, muss dieser in der Navigationsleiste auf Covid-19 Daten klicken. Es öffnet sich eine neue Seite. Zu sehen ist eine Liste in Form einer Tabelle, mit den personenbezogenen Daten der erfassten Kunden.
Der Benutzer hat auf dieser Seite folgenden Optionen:

  • Personenbezogenen Daten als CSV-Datei exportieren durch das Klicken auf CSV Export
  • Personenbezogenen Daten als PDF-Date exportieren durch das Klicken auf PDF Export
  • Personenbezogenen Daten als PDF-Datei direkt im Browser anzeigen lassen durch das Klicken auf PDF anzeigen
  • Personenbezogenen Daten durch das Klicken auf (Mülltonnen-Symbol). Daraufhin wird der Benutzer auf die Löschen Seite weitergeleitet, wo der Benutzer die Daten der Person vor dem löschen überprüfen kann. Durch eine Klick auf Löschen wird der Eintrag aus der Liste gelöscht. Ein Klick auf Abbrechen beendet den Vorgang und der Benutzer wird zurück auf die Übersicht der Covid-19 Daten weitergeleitet
  • Nach bestimmten Personen suchen, in dem die Filter-Funktion eingesetzt wird

Template Auswahl, anzeigen der Menükarte & Export des QR-Codes

Der Benutzer hat die Möglichkeit, das Design der Menükarte jederzeit zu verändern. Hierfür kann er auf dem Dashboard unter Template-Auswahl durch eine Dropwdown-Liste sich eines von drei Templates aussuchen. Durch klicken auf Speichern übernimmt das System die Änderung und es erscheint eine Meldung, dass das Template – und somit das Design der Menükarte erfolgreich geändert wurde.

Der Benutzer kann sich nun die Menükarte durch klicken auf Zur Menükarte in einem neuen Browser-Tab anzeigen lassen. Es wird empfohlen sich die Menükarte im neuen Browser-Tab in der mobilen Ansicht anzusehen. Mit einem Rechtsklick auf dem Tab klickt der Benutzer in der Liste auf Untersuchen und innerhalb der Entwicklungskonsole auf das Smartphone-Tablet-Sybmol. Dadurch erhält der Benutzer die mobile Ansicht des Menükarte.

Kunden können die Menükarte durch das Scannen des QR-Codes über den Browser im Smartphone erreichen. Diese kann der Benutzer durch Klicken auf QR-Code generieren als PDF-Datei exportieren, ausdrucken und im Lokal den Kunden zur Verfügung stellen.
